TrackJS
===============

A [TrackJS](http://www.trackjs.com) Bower module. You'll need a [TrackJS Account](http://trackjs.com/).

##Install

####Bower

	bower install trackjs --save

####Activate account

[Sign up for a TrackJS account](https://my.trackjs.com/signup).
